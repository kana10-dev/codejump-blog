### 3. CSSによる繊細なスタイリングとアニメーション

- **レイアウト制御**: `justify-content: flex-start;` による要素の緻密な配置に加え、`border-width` や `border-color` のプロパティを組み合わせた視覚的なデザイン構築を実践した。
